Notes

Prototype 1 Difficulties

The drop down menus are simple to use for the user to navigate between several pages of this website. However, it was one of the harder parts for the project to code. It took me over six hours to figure out why the sub menus were not working correctly. I finally figured out the answer which was to move the sub menu in the z-axis over the body content of the page.

Prototype 2 Improvements

I made the menu navigation bar a bit better, adding in an ease-in transition. I also formated the navbar to work on tablet screen sizes (now you have the full computer window view of this website, tablet view, and phone view with a dynamic navigation bar). Visitors are now able to see where they currently are in the navigatin bar. I fixed some issues where div boxes overlapped with other div boxes when the screen size got smaller. Images are now optimized for file size. There is now less space at the top of the page as well.

My dynamics page in the Contacts page. When the user clicks on the "Send Me Comments" the page will dynamically change to a another page that has a fill-out-form in which the viewer can send me comments.

Miscellany

JQUERY and JavaScript was used in my imported lightbox effects for pictures and video files on my website. JavaScript was also used to change the description box under the images in the digital and tradtional galleries.

For my above and beyond, I created the website logo from scratch, imported/used HTML and CSS code for light boxes from other web sources. I used Sketchfab to display my 3D model in real-time. Lastly, I used a software called IceCream Screen Recorder to record videos of my game being played.
